About This Project

This is a Tennis Swing Analysis System which is a real-time sports analytics platform developed for the IDEA LAB SEE. It leverages a wearable sensor device to capture accelerometer and gyroscope data, which is then processed by a sophisticated machine learning model to classify tennis strokes instantly.

The Technology

The core of our prediction engine is a Stacking Classifier, an advanced ensemble method that combines the strengths of several different models to achieve higher accuracy and robustness. This meta-model makes a final prediction based on the outputs of the following base models:

K-Nearest Neighbors (KNN)

Classifies a data point based on the majority class among its 'k' closest neighbors in the feature space.

Logistic Regression

A statistical model that uses a logistic function to model the probability of a certain class or event existing.

Random Forest Classifier

An ensemble of decision trees that improves accuracy and controls over-fitting by averaging multiple results.

Support Vector Classifier (SVC)

Finds an optimal hyperplane in an N-dimensional space that distinctly classifies the data points.
